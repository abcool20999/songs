### Song Catalogue App

This is an app that serves as a song catalogue, displaying trending latest songs along with the artists who performed them and the awards the songs have won.

#### Implemented Features:
- **Authentication:** Authentication system has been implemented.
- **CRUD Functions:** CRUD (Create, Read, Update, Delete) functions have been implemented for both songs and awards.
- **One-to-Many Relationship:** A One-to-Many relationship between songs and awards has been implemented, although there were some difficulties and errors during implementation.
